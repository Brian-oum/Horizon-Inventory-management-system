Upload IoT Devices via Excel

{% csrf_token %}
Your Excel file must include the following columns (case-insensitive):
  • Supplier ID – Required, must match an existing supplier’s ID
  • Product ID – Optional, defaults to IMEI if blank
  • Name – Optional, device name (e.g., FMB220, Sensor X)
  • IMEI No – Required, can contain one or multiple IMEIs separated by commas, must be unique
  • Serial No – Optional, applied to all IMEIs if multiple are provided
  • Category – e.g., Dashcam, Tracker, Sensor
  • Description – Device description
  • Selling Price (USD) – Optional
  • Selling Price (KSH) – Optional
  • Selling Price (TSH) – Optional
  • Status – Must be one of: available, issued, returned, faulty
  • Quantity – Optional; if blank, system auto-calculates based on number of IMEIs in that row
Notes:
- Supplier ID must exist in the system. Devices without a matching supplier will still be uploaded but shown with a warning.
- At least one price column (USD, KSH, or TSH) may be filled.
- IMEI numbers must always be unique.
- If multiple IMEIs are provided in one row, the system will create separate devices for each IMEI and set Quantity to their count.
